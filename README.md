# CapGemini


Configurez les variables d'environnements (.env)
de cette façon 
```
    POSTGRES_USER=XXXXX
    POSTGRES_PASSWORD=XXXX
    POSTGRES_DB=XXXX

```

Vérifiez bien que vous avez toutes les librairies en faisant la commande : 
```npm install 
```
Ensuite lancez la commande suivante : 
```
docker-compose up

```
